# 精准抗感染专项系列会议信息管理系统

用于线上医学学术会议的策划、信息收集、预览确认、导出归档和后台管理。

## 功能概览

- 首页项目介绍与子项目入口
- 新建会议与编辑会议
- 大区管理：东区 / 西区 / 北区 / 南区
- 讲题动态配置：每场会议支持 2 / 3 / 4 个讲题
- 角色化人员录入：主席、主持、讲者、讨论嘉宾
- 历史会议检索、筛选、复制、删除、详情查看
- Excel / PDF / 打印导出
- 本地存储保存会议记录

## 技术栈

- React 19
- Vite 8
- React Router
- XLSX
- jsPDF
- html2canvas

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址：

```text
http://localhost:4173/
```

## 生产构建

```bash
npm run build
```
